# tinyLlamaChat
Web UI for for the tinyLlama-chat model
---

## Features
 - Chatting with tinyLlama chat model
 - Extremely fast throughput through vLLM
 - simple UI/UX 
 - Configarble llm values (Temp, top p, max tokens)
